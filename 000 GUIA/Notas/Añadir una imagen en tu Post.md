Para añadir una imagen a tu post (y que se pueda ver en tu página web) tienes varios caminos

## Featured 
La primera es la imagen "featured". Cuando quieres que una imagen sea la portada de tu nota debes agregarla a la Carpeta de tu Post y ponerle de nombre featured.

## Markdown
Puedes añadir una foto usando markdown como lo harías normalmente, así:
```
![Texto que quieras poner](mi-imagen.jpg)
```

## Html
Si quieres controlar las dimensiones de la imagen puedes usar html, asi
```
<img src="mi-imagen.jpg" width="300">
```
